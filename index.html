HTML

Structure Overview

Elements

HTML (hypertext markup language) is made up of elements. HTML elements generally contain two tags--an opening tag and a closing tag. Elements tell the browser something about the information in the opening and closing tags.

Tree Structure

HTML is organized into a family tree structure. HTML elements can have parents, grandparents, siblings, children, grandchildren, etc.

HTML Tags

Tag Quick Reference Table

Tag Name Description
<a> Anchor Creates a link to another page or to a location in the current page.
<abbr> Abbreviation Indicates an acronym or abbreviation of a longer word or phrase.
<acronym> Acornym Creates text that will be displayed when hovered over.
<audio> Audio Represents an interface for adding audio content to the page.
<blockquote> Blockquote Represents a section of a document that contains a longer quotation, usually spanning multiple lines.
<p> Paragraph Contains and displays a block of text that defines a paragraph.

CSS

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.
